# Backend

Express + TypeScript backend server for ICPC Website.

## Setup

Install dependencies:
```bash
npm install
```

## Development

Run the development server:
```bash
npm run dev
```

## Build

Build for production:
```bash
npm run build
```

## Start

Run the production server:
```bash
npm start
```

The server will run on `http://localhost:5000` by default.
